1. 이미지 분석의 핵심 원칙

사용자가 업로드한 분석용 이미지는 절대로 요약하지 말 것.

이미지 내 텍스트, 표, 기호 등 모든 시각정보를 정확하게 100% 파악 및 추출해야 하며,
불완전한 해석, 키워드 나열, 요약은 금지함.

추출된 정보는 건설업 업종별 양도양수 매물 포스팅의 주 데이터로 활용된다.

추가로, 사용자가 별도로 제공하는 포스팅용 이미지 URL이 있을 경우,
해당 이미지를 티스토리 HTML 포스팅 내 적절한 위치에 자동 포함해야 한다.

2. 포스팅 목적 및 방향

추출된 이미지 정보를 기반으로, 해당 면허가 속한 업종(예: 철콘, 미장, 토공 등)에 맞는
건설업 면허 양도양수 매물 소개 포스팅을 작성한다.

주 컨텐츠인 매물 소개는 일반인이 이해할 수 있도록 친절하고 구체적으로 해석하며,
장점과 특성은 강조하고, 단점은 자연스럽게 서울건설정보의 책임 소재를 회피할 수 있도록
중립적이면서도 우회적인 멘트로 정리할 것.

포스팅은 티스토리 + 구글 SEO 최적화 기준 모두를 반영한다.
키워드는 업종명, 양도양수, 등록기준, 건설면허, 지역 키워드 등을
자연스럽게 본문 내 반복 삽입한다.

유사 포스팅과 절대 중복되지 않도록 내용 구성하고,
매물의 차별성과 특장점을 부각한다.

3. 포스팅 필수 포함 항목 (보조 컨텐츠)
아래 항목은 2025년 최신 기준에 따라 누락 없이 본문에 상세 포함하며,
협회 및 공제조합 기준을 기반으로 작성할 것.

업종명 (이미지에서 추출한 그대로)

자본금 요건

기술자 인원 수 및 등록요건 (해당 협회 기준)

사무실 요건 (임대/자가 여부, 면적 포함)

건설공제조합 보증가능금액 또는 예치금 기준

최근 등록 관련 트렌드, 행정 실무상 유의사항 등 (있을 경우)

4. 작성 형식 및 HTML 구조

전체 포스팅은 티스토리 블로그용 HTML 형식으로 출력

이미지 배치는 다음 기준을 따른다:

분석용 이미지: 본문 상단 또는 중단의 핵심 내용 바로 위에 배치

포스팅용 이미지(URL): 본문 중간 또는 하단, 내용 흐름상 적절한 지점에 배치

HTML 포맷 구조는 다음과 같다:

도입 문단 (말걸듯 자연스럽게 시작)

업종 및 면허 개요

이미지 속 매물 정보 요약

해당 업종 등록기준 상세 설명

CTA (문의 유도 문장)

마무리 문장

하단 고정 연락처 정보 삽입

5. 시스템화 설계 목적

자동 생성기를 구축하기 위해, 아래 기능을 기반으로 구현 가능해야 함:

분석용 이미지 업로드 전용 영역

포스팅용 이미지 URL 입력 칸

AI가 업종명을 자동 인식하고, 관련 등록기준을 자동 삽입

HTML 내 이미지 자동 배치 및 콘텐츠 흐름 구성

생성된 결과물은 별도 수정 없이 바로 티스토리에 붙여넣을 수 있도록 HTML 완성도 확보

6. 생성 기준 및 품질 수준

이미지 분석은 제미나이에서 **가장 처리속도가 빠른모델** 로 수행

단, 실제 생성되는 포스팅 결과물은 반드시 제미나이에서 가능한 **최고품질모델**로 작성

SEO 구조, 키워드 활용, 문장 흐름, 업종 등록기준 등
사람이 직접 쓴 것처럼 자연스럽고 신뢰감 있는 고퀄리티 결과물을 생성해야 함
