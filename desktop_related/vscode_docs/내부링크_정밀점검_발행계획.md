## 4. 발행 시 주의사항

1. **카카오톡 URL**: 모든 파일의 상단 CTA에 sXXXXXXX 플레이스홀더를 실제 URL로 교체  플레이스홀더 -> 실제 URL 교체
2. **이미지**: [이미지 위치 1~4] 표기 위치에 실제 이미지 삽입 (에디터 alt 속성만 입력, 본문 텍스트 노출 금지)
3. **미발행 글 제목**: 허브에서 URL 없이 제목만 기재된 글은, 해당 글 발행 시 URL 추가
4. **기존 글 비공개 처리**: 대체 대상 구글(舊글)은 새 글 발행 확인 후 비공개 전환
5. **네이버 에디터**: seoulmna.co.kr 외부 링크는 새 탭으로 열리도록 설정
