# 특허 패키지 초안 A: AI 양도가 산정 계산기 (2026-03-05)

## 1. 출원 목표
- 출원명(초안): `건설업 양도양수 거래에서 다변수 리스크를 반영한 양도가 산정 및 신뢰도 제공 방법`
- 범위: 양도양수 산정 엔진 + 설명가능 결과 생성 + 상담 연계 트리거.
- 목표: 단순 시세 평균이 아닌, 입력변수의 불확실성과 업종 특수성을 반영한 산정 절차를 권리화.

## 2. 해결하려는 문제
- 기존 방식의 한계:
1. 업종/재무/행정이력/신용을 동시에 반영하지 못함.
2. 산정 과정이 블랙박스라 사용자 신뢰 형성에 취약.
3. 운영 관점에서 멀티테넌트/보안/비용 제어가 분리되어 재현성이 낮음.

## 3. 핵심 발명 포인트(요약)
1. 입력 정규화:
- 업종 문자열/연도 재무값/행정이력/신용수준을 표준 형식으로 변환.
2. 유사사례 탐색 + 가중 보정:
- 유사도(문자열·재무·추세)를 결합해 이웃 사례군을 구성하고 중심값/범위를 산정.
3. 신뢰도 스코어:
- 데이터 충족도, 이웃 밀집도, 입력 일관성을 기반으로 신뢰도 수치화.
4. 운영 안전장치:
- tenant 정책/보안키/차단키/요청로그를 산정 파이프라인에 결합하여 상용 운영 일관성 확보.

## 4. 독립항 초안(기술 관점)
1. 방법 청구항(초안)
- 건설업 양도양수 거래의 양도가를 산정하는 컴퓨터 구현 방법에 있어서,
  - 사용자 입력을 업종/재무/행정/신용의 표준 벡터로 정규화하고,
  - 정규화 벡터와 기준 데이터셋 간의 복수 유사도 함수를 결합해 유사 사례군을 선택하며,
  - 선택된 사례군에 대해 가중 중심값 및 변동범위를 산정하고,
  - 입력 결측 및 데이터 품질 지표를 반영한 신뢰도 지표를 생성하여,
  - 상기 중심값, 변동범위, 신뢰도를 함께 출력하는 것을 특징으로 하는 방법.

2. 시스템 청구항(초안)
- 상기 방법을 수행하는 입력 정규화 모듈, 유사도 평가 모듈, 산정 모듈, 신뢰도 산정 모듈, 결과 제공 모듈을 포함하는 시스템.

## 5. 종속항 후보
1. 유사도 함수가 문자 bigram 유사도 + 재무 상대근접도 + 연도별 추세 유사도를 결합하는 구성.
2. 행정처분/신용등급 입력이 산정 범위 가중치에 추가 반영되는 구성.
3. 결과 출력 시 신뢰도 구간에 따라 상담 연계 경로를 자동 분기하는 구성.
4. tenant별 기능 제한 및 차단키 정책이 결과 제공 경로에 적용되는 구성.

## 8. 변리사 전달 체크포인트
1. 선행기술 대비 차별점:
- 단순 시세 조회가 아닌 `다중 유사도 + 신뢰도 + 운영정책 결합`.
2. 권리 범위 전략:
- 방법 + 시스템 + 컴퓨터 판독가능 기록매체 3종 세트.
3. 회피 설계 방지:
- 유사도 함수 유형이 달라도 “복수 유사도 결합 + 신뢰도 출력”을 포섭하도록 표현.
