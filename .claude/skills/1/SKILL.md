# Standing Directive "1" — 자율 개발 총괄 실행

이 스킬이 호출되면 아래 전체 directive를 즉시 실행합니다. 질문 없이 자율 진행합니다.

## 목표 (4개 병행)
1. **AI 양도가 산정 시스템** 개발
2. **AI 인허가 사전검토 시스템** 개발
3. 2가지 시스템 탑재를 위한 **seoulmna.kr 플랫폼화** (또는 더 좋은 대안)
4. 위 2가지 **시스템 특허** 준비

## 작업 범위
- 위 내용과 관련없는 파일은 배제 (다른 곳에 사용되는 파일은 수정하지 않고 무시)
- **H:\auto** = CORE 작업 영역 (AI 시스템 + 플랫폼 + 특허 + 인프라)
- **H:\ALL** = NON-CORE (블로그·CRM 등 — 참조만, 수정 금지)
- **.co.kr 관련 파일 절대 수정 금지** (Cursor에서 별도 진행 중)
- **.kr 플랫폼은 H:\auto에서만 진행** (H:\ALL과 혼동 방지)

## 실행 원칙
- CODEX를 1000% 활용하여 작업, Gemini CLI 활용 여부도 정밀 검토
- **브레인스토밍 → 기획 → 수행 → 검수 → 최종 완료** 자율 반복
- 브레인스토밍/비판적 사고/문제 해결 반복하여 개선점 스스로 되묻기
- 요청하지 않은 사항이라도 긍정적 영향 예상되면 반영
- 총괄 진행은 AI가 담당, 사용자는 엔터만 누르는 역할
- 정밀하고 정교하게, 디테일을 놓치지 말고 모든 업무를 혼자 수행
