## 플랫폼 전환 체크리스트
### 현재 상태 (블로그)
- WordPress/Astra 기본 그리드 레이아웃
- 블로그 글 나열 — 서비스 플랫폼 느낌 없음
- 히어로/CTA/서비스 소개 섹션 없음
- AI 시스템 진입점 없음

### 목표 상태 (전문 플랫폼)
- [ ] **히어로 섹션**: 건설업 AI 전문 플랫폼 메인 비주얼 + 핵심 CTA
- [ ] **서비스 카드**: AI 양도가 산정 / AI 인허가 사전검토 2개 서비스 카드
- [ ] **신뢰 시그널**: 분석 건수, 정확도, 데이터 소스, 행정사 인증 배지
- [ ] **가격 비교**: 기존 행정사 vs 브로커 vs 서울건설정보 (admini.kr 참조)
- [ ] **기능 소개**: 핵심 기능 4~6개 아이콘 + 설명 섹션
- [ ] **고객 후기/실적**: 실시간 활동 피드 또는 통계 카운터
- [ ] **지식 콘텐츠**: 기존 블로그 글을 "건설업 지식" 섹션으로 재배치
- [ ] **글로벌 네비게이션**: 홈/양도가산정/인허가검토/지식/상담/매물시장
- [ ] **푸터**: 회사정보, 연락처, 인증, 약관
- [ ] **모바일 최적화**: 반응형 + 터치 최적화 (44px 터치 타겟)
- [ ] **다크 모드**: admini.kr 참조 (선택)
